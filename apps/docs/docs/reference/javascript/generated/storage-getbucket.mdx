检索现有存储桶的详细信息。

```js
const { data, error } = await supabase
  .storage
  .getBucket('avatars')
```

## 注意事项

- 需要RLS策略权限:
  - `buckets`表权限: `选择'.
  - `objects`表的权限：无
- 请参考[存储指南](/docs/guides/storage#access-control)中关于访问控制的工作方式。










## 实例

### 获取桶



```js
const { data, error } = await supabase
  .storage
  .getBucket('avatars')
```
