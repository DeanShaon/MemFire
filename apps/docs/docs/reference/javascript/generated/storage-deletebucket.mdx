删除一个现有的桶。一个水桶不能在其内部存在对象的情况下被删除。你必须首先`empty()`该桶。

```js
const { data, error } = await supabase
  .storage
  .deleteBucket('avatars')
```

## 注意事项

- 需要RLS策略权限:
  - `buckets`表的权限: `选择'和`删除'.
  - `objects`表的权限：无
- 请参考[存储指南](/docs/guides/storage#access-control)中关于访问控制的工作方式。
