取消订阅并从Realtime客户端删除Realtime频道。

```js
supabase.removeChannel(myChannel)
```

## 注意事项

- 如果你在监听Postgres的变化，移除一个通道是保持你项目的Realtime服务以及数据库性能的一个好方法。Supabase会在客户端断开连接后的30秒内自动处理清理工作，但未使用的通道可能会因为同时订阅了更多的客户端而导致性能下降。










## 示例

### 移除一个频道



```js
supabase.removeChannel(myChannel)
```
