列出桶内的所有文件。

```js
const { data, error } = await supabase
  .storage
  .from('avatars')
  .list('folder', {
    limit: 100,
    offset: 0,
    sortBy: { column: 'name', order: 'asc' },
  })
```

## 注意事项

- 需要RLS策略权限:
  - `buckets`表的权限: 无
  - `objects`表的权限: `select`.
- 请参考[存储指南](/docs/guides/storage#access-control)中关于访问控制的工作方式。










## 示例

### 在一个桶中列出文件



```js
const { data, error } = await supabase
  .storage
  .from('avatars')
  .list('folder', {
    limit: 100,
    offset: 0,
    sortBy: { column: 'name', order: 'asc' },
  })
```

### 在一个桶中搜索文件



```js
const { data, error } = await supabase
  .storage
  .from('avatars')
  .list('folder', {
    limit: 100,
    offset: 0,
    sortBy: { column: 'name', order: 'asc' },
    search: 'jon'
  })
```
