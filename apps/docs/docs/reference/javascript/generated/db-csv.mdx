返回`data`为CSV格式的字符串。

```ts
const { data, error } = await supabase
  .from('countries')
  .select()
  .csv()
```


















## 示例

### 以CSV格式返回数据

默认情况下，数据以JSON格式返回，但也可以以逗号分隔值的形式返回。
