将`data`作为一个对象而不是一个对象的数组返回。

查询结果必须是0或1行（例如，使用`.limit(1)`），否则这将返回一个错误。

```ts
const { data, error } = await supabase
  .from('countries')
  .select()
  .eq('name', 'Singapore')
  .maybeSingle()
```
