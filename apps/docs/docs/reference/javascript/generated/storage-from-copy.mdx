将一个现有的文件复制到同一桶中的新路径。

```js
const { data, error } = await supabase
  .storage
  .from('avatars')
  .copy('public/avatar1.png', 'private/avatar2.png')
```

## 注意事项

- 需要RLS策略权限:
  - `buckets`表的权限: 无
  - `objects`表的权限: `插入'和`选择'.
- 请参考[存储指南](/docs/guides/storage#access-control)中关于访问控制的工作方式。










## 实例

### 复制文件



```js
const { data, error } = await supabase

  .from('avatars')

```
