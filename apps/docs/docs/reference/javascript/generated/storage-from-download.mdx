下载文件。

```js
const { data, error } = await supabase
  .storage
  .from('avatars')
  .download('folder/avatar1.png')
```

## 注意事项

- 需要RLS策略权限:
  - `buckets`表的权限: 无
  - `objects`表的权限: `select`.
- 请参考[存储指南](/docs/guides/storage#access-control)中关于访问控制的工作方式。
