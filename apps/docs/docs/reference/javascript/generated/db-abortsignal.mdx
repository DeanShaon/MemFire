设置获取请求的AbortSignal。

```ts
const ac = new AbortController()
ac.abort()
const { data, error } = await supabase
  .from('very_big_table')
  .select()
  .abortSignal(ac.signal)
```

## 实例

### 终止执行中的请求

你可以使用一个[`AbortController`](https://developer.mozilla.org/en-US/docs/Web/API/AbortController)来中止请求。请注意，`status`和`statusText`对中止的请求没有任何意义，因为请求没有得到满足。
