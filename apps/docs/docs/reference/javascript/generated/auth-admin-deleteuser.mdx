删除一个用户。需要一个`service_role`键。

```js
const { data, error } = await supabase.auth.admin.deleteUser(
  '715ed5db-f090-4b8c-a067-640ecee36aa0'
)
```

## 注释

- `deleteUser()`方法需要用户的ID，它映射到`auth.users.id`列。










## 示例

### 移除一个用户



```js
const { data, error } = await supabase.auth.admin.deleteUser(
  '715ed5db-f090-4b8c-a067-640ecee36aa0'
)
```
