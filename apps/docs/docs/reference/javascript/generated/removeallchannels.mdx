取消订阅并从Realtime客户端删除所有Realtime频道。

```js
supabase.removeAllChannels()
```






## 注意事项

- 如果你在监听Postgres的变化，移除通道是保持你项目的实时服务以及数据库性能的一个好方法。Supabase会在客户端断开连接后的30秒内自动处理清理工作，但是当更多的客户端同时订阅时，未使用的通道可能导致性能下降。










## 例子

### 移除所有通道



```js
supabase.removeAllChannels()
```
