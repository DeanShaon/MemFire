通过ID获取用户。

```js
const { data, error } = await supabase.auth.admin.getUserById(1)
```

## 注释

- 根据用户的ID从数据库中获取用户对象。
- `getUserById()`方法需要用户的id，它映射到`auth.users.id`列。










## 示例

### 使用access_token获取用户对象 jwt.



```js
const { data, error } = await supabase.auth.admin.getUserById(1)
```
