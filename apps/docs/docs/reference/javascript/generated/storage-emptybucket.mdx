移除单个桶内的所有对象。

```js
const { data, error } = await supabase
  .storage
  .emptyBucket('avatars')
```

## 注意事项

- 需要RLS策略权限:
  - `buckets`表权限: `选择'.
  - `objects`表的权限: `select`和`delete`.
- 请参考[存储指南](/docs/guides/storage#access-control)关于访问控制的工作方式










## 例子

### 空桶

const { data, error } = await supabase
