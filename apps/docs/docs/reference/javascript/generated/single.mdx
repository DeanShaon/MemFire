将`data`作为一个对象而不是一个对象的数组返回。

查询结果必须是一行（例如，使用`.limit(1)`），否则会返回一个错误。

```ts
const { data, error } = await supabase
  .from('countries')
  .select('name')
  .limit(1)
  .single()
```
