Deletes files within the same bucket


```dart
final res = await supabase
  .storage
  .from('avatars')
  .remove(['avatar1.png']);
```






## Notes

- Policy permissions required:
  - `buckets` permissions: none 
  - `objects` permissions: `delete` and `select`










## Examples
