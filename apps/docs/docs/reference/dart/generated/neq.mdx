查找所有在所述 `column`上的值与指定的 `value`不一致的记录。


```dart
final data = await supabase
  .from('cities')
  .select('name, country_id')
  .neq('name', 'The shire');
```


















## Examples

### 使用 `select()`



```dart
final data = await supabase
  .from('cities')
  .select('name, country_id')
  .neq('name', 'The shire');
```

### 使用 `update()`



```dart
final data = await supabase
  .from('cities')
  .update({ 'name': 'Mordor' })
  .neq('name', 'San Francisco');
```

### 使用 `delete()`



```dart
final data = await supabase
  .from('cities')
  .delete()
  .neq('name', 'Mordor');
```

### 使用 `rpc()`



```dart
// Only valid if the Stored Procedure returns a table type.
final data = await supabase
  .rpc('echo_all_cities')
  .neq('name', 'Lagos');
```
