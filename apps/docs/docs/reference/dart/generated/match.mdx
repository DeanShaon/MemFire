查找所有列与指定的`query`对象相匹配的记录。


```dart
final data = await supabase
  .from('cities')
  .select('name, country_id')
  .match({'name': 'Beijing', 'country_id': 156});
```


















## Examples

### 使用 `select()`



```dart
final data = await supabase
  .from('cities')
  .select('name, country_id')
  .match({'name': 'Beijing', 'country_id': 156});
```

### 使用 `update()`



```dart
final data = await supabase
  .from('cities')
  .update({ 'name': 'Mordor' })
  .match({'name': 'Beijing', 'country_id': 156});
```

### 使用 `delete()`



```dart
final data = await supabase
  .from('cities')
  .delete()
  .match({'name': 'Beijing', 'country_id': 156});
```

### 使用 `rpc()`



```dart
// Only valid if the Stored Procedure returns a table type.
final data = await supabase
  .rpc('echo_all_cities')
  .match({'name': 'Beijing', 'country_id': 156});
```
