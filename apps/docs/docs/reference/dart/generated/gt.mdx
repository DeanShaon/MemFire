查找所有在所述`column`上的值大于指定`value`的记录。


```dart
final data = await supabase
  .from('cities')
  .select('name, country_id')
  .gt('country_id', 250);
```


















## Examples

### 使用 `select()`



```dart
final data = await supabase
  .from('cities')
  .select('name, country_id')
  .gt('country_id', 250);
```

### 使用 `update()`



```dart
final data = await supabase
  .from('cities')
  .update({ 'name': 'Mordor' })
  .gt('country_id', 250);
```

### 使用 `delete()`



```dart
final data = await supabase
  .from('cities')
  .delete()
  .gt('country_id', 250);
```

### 使用 `rpc()`



```dart
// Only valid if the Stored Procedure returns a table type.
final data = await supabase
  .rpc('echo_all_cities')
  .gt('country_id', 250);
```
