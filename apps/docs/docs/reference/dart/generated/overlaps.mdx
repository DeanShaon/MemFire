## Examples

### 使用 `select()`



```dart
final data = await supabase
  .from('countries')
  .select('name, id, main_exports')
  .overlaps('main_exports', ['computers', 'minerals']);
```

### 使用 `update()`



```dart
final data = await supabase
  .from('countries')
  .update({ 'name': 'Mordor' })
  .overlaps('main_exports', ['computers', 'minerals']);
```

### 使用 `delete()`



```dart
final data = await supabase
  .from('countries')
  .delete()
  .overlaps('main_exports', ['computers', 'minerals']);
```

### 使用 `rpc()`



```dart
// Only valid if the Stored Procedure returns a table type.
final data = await supabase
  .rpc('echo_all_countries')
  .overlaps('main_exports', ['computers', 'minerals']);
```
