对表进行DELETE操作。


```dart
final data = await supabase
  .from('cities')
  .delete()
  .match({ 'id': 666 });
```






## Notes

- `delete()`应该总是与[Filters](/docs/reference/dart/using-filters)结合起来，以锁定你想删除的项目。










## Examples

### 删除记录



```dart
final data = await supabase
  .from('cities')
  .delete()
  .match({ 'id': 666 });
```
