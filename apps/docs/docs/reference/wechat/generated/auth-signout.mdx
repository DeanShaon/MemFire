在浏览器上下文中，`signOut()`将把登录的用户从浏览器会话中移出
并注销 - 从本地存储中删除所有项目，然后触发一个`"SIGNED_OUT"`事件。

对于服务器端的管理，你可以通过将用户的JWT传递给`auth.api.signOut(JWT: string)`来撤销一个用户的所有刷新令牌。
在用户的访问令牌jwt过期之前，没有办法撤销它。出于这个原因，建议在jwt上设置一个较短的到期时间。

```js
const { error } = await supabase.auth.signOut()
```






## 注释

- 为了使用`signOut()`方法，用户需要先登录。










## 例子

### 退出登录



```js
const { error } = await supabase.auth.signOut()
```
