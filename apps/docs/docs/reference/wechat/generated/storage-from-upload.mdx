将一个文件上传到一个现有的桶。

```js
const avatarFile = event.target.files[0]
const { data, error } = await supabase
  .storage
  .from('avatars')
  .upload('public/avatar1.png', avatarFile, {
    cacheControl: '3600',
    upsert: false
  })
```

## 注意事项

- 需要RLS策略权限:
  - `buckets`表的权限: 没有
  - `objects`表的权限: `insert'.
- 请参考[Storage guide](/docs/guides/storage#access-control)关于访问控制的工作方式
- 对于React Native，使用`Blob`、`File`或`FormData`都不能按预期工作。使用`ArrayBuffer`从base64文件数据上传文件，而不是使用`ArrayBuffer`，见以下例子。










## 实例

### 上传文件



```js
const avatarFile = event.target.files[0]
const { data, error } = await supabase
  .storage
  .from('avatars')
  .upload('public/avatar1.png', avatarFile, {
    cacheControl: '3600',
    upsert: false
  })
```

### 使用`ArrayBuffer`从base64文件数据上传文件



```js
import { decode } from 'base64-arraybuffer'

const { data, error } = await supabase
  .storage
  .from('avatars')
  .upload('public/avatar1.png', decode('base64FileData'), {
    contentType: 'image/png'
  })
```
