获取一个用户列表。

这个函数应该只在服务器上调用。不要在浏览器中暴露你的`service_role`键。

```js
const { data: { users }, error } = await supabase.auth.admin.listUsers()
```


















## 例子

### 获得完整的用户列表。



```js
const { data: { users }, error } = await supabase.auth.admin.listUsers()
```
