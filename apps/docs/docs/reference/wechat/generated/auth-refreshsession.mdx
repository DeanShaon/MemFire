返回一个新的会话，无论是否过期。
接收一个可选的当前会话。如果没有传入，那么refreshSession()将试图从getSession()中获取它。
如果当前会话的刷新令牌无效，将抛出一个错误。

```js
const { data: { user, session }, error } = await supabase.auth.refreshSession()
```

## 注释

- 这个方法将刷新会话，无论当前会话是否过期。
- 这两个例子都是从`data'解构`user'和`session'。这不是必须的，所以`const { data, error } =`也是有效的。










## 示例

### 使用当前会话刷新会话



```js
const { data: { user, session }, error } = await supabase.auth.refreshSession()
```

### 使用传入的会话刷新会话



```js
const { data: { user, session }, error } = await supabase.auth.refreshSession({ refresh_token })
```
